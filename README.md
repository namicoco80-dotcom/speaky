 # ⚡ SPEAKY — 나만의 영어 표현 노트

## 🚀 실행 방법

**`index.html` 파일을 더블클릭하면 바로 시작됩니다.**

> Chrome 또는 Edge 브라우저를 권장해요.
> Safari, Firefox도 동작하지만 TTS 품질이 다를 수 있어요.

---

## 📱 홈화면에 추가하기 (선택사항)

브라우저 주소창 오른쪽 설치 아이콘(⊕)을 누르면  
앱처럼 설치해서 사용할 수 있어요.

---

## 💾 내 데이터는 어디에?

모든 문장·학습기록은 **내 컴퓨터 브라우저**에만 저장돼요.  
인터넷 연결 없이도 사용 가능합니다.  
(한글 자동번역만 인터넷 필요)

---

## 🔒 백업 방법

설정 → 데이터 관리 → **백업(내보내기)**  
→ JSON 파일로 저장해두세요.

새 컴퓨터로 옮길 때:  
설정 → 데이터 관리 → **복구(가져오기)**
